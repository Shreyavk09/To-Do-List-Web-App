# To-Do List Web App

A simple, interactive To-Do List built using **HTML, CSS, and Vanilla JavaScript**.

## Features
- Add, complete, and delete tasks dynamically
- Instant UI updates without reloading
- Clean and responsive interface

## Tools Used
- HTML5
- CSS3
- JavaScript (ES6)
- VS Code + Live Server
